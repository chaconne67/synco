# t16: 조직 관리 템플릿 구현

> **Phase:** 2단계 — 통합 설정 + 조직 관리
> **선행 조건:** t15 (조직 관리 뷰 + URL)

---

## 배경

t15에서 조직 관리의 URL, 뷰, 테스트를 구성했다. 이 태스크에서는 실제 렌더링할 템플릿을 구현한다.

---

## 요구사항

### org_base.html 구조

설정 페이지와 동일한 탭 바 + 콘텐츠 컨테이너 패턴:

```
org_base.html
  ├── org_tab_bar.html  (조직 정보 | 멤버 관리 | 초대코드)
  └── #org-content
       └── (각 탭 파셜)
```

### 탭 바 (`org_tab_bar.html`)

- 3개 탭 버튼: 조직 정보, 멤버 관리, 초대코드
- `hx-get` + `hx-target="#org-content"` + `hx-push-url="true"`
- 설정 탭과 동일한 스타일 패턴

### 조직 정보 탭 (`org_info.html`)

- 조직명 입력 필드 (수정 가능)
- 로고 업로드 (기존 로고 미리보기 포함)
- 플랜 표시 (읽기 전용, 뱃지 스타일)
- DB 공유 상태 표시 (읽기 전용)
- 저장 버튼 (HTMX POST)

### 멤버 관리 탭 (`org_members.html`)

멤버 카드 목록:
- 이름 / 역할 뱃지 / 상태 뱃지 / 가입일
- 승인대기 멤버: 승인/거절 버튼
- 활성 멤버 (owner 제외): 역할 변경 드롭다운 + 제거 버튼
- 제거 시 확인 다이얼로그 (`hx-confirm`)
- 상태 메시지 표시 (승인/거절/변경/제거 후)

### 초대코드 탭 (`org_invites.html`)

- 생성 폼: 역할/최대 사용 횟수/만료일 3열 그리드
- 코드 목록: 데스크톱 테이블 + 모바일 카드 반응형
- 코드 복사: `navigator.clipboard.writeText()` + 토스트 메시지
- 비활성화 버튼 (HTMX POST)
