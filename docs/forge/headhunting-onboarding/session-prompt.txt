plan-forge-batch 오케스트레이터의 세션 체인 중 하나다. 이 오케스트레이터는 복수의 할일을 순차적으로 담금질→구현하되, 각 작업 단위마다 새 CLI 세션을 시작하여 컨텍스트 윈도우 고갈을 방지한다.

## 실행 절차
1. CLAUDE.md를 읽어 프로젝트 컨텍스트를 파악하라
2. plan-forge-batch 스킬을 호출하라 (Skill 도구 사용). 스킬이 forge-progress.json을 읽고 자동으로 Continue 모드로 진입한다. 스킬이 1작업단위 실행 → forge-progress.json 업데이트 → git commit → 다음 세션 spawn(또는 종료)까지 모두 처리한다
3. 스킬이 종료되면 이 세션도 종료한다. 다음 세션의 spawn은 스킬 내부에서 이미 처리되었으므로, 이 세션에서 추가 작업을 하지 않는다
