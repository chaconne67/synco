## Journey 1: FC (Supply Side)

```
발견 → 가입 → 활성화 → CEO 전환 → 수익 → 확산
```

| Stage | 행동 | 감정 | 터치포인트 | 드롭오프 리스크 |
|-------|------|------|-----------|----------------|
| **발견** | ① **GA 법인 파트너십 체결 → GA가 소속 FC에게 synco(FC 모드) 도입 안내** (주요 경로) ② 동료 FC 또는 지점장이 소개 | 호기심 + 의심 ("또 CRM?") | GA 공지, 대면 소개, 카톡 공유 | "기존 CRM과 뭐가 다른데" |
| **가입** | synco(FC 모드) 설치, 기본 정보 입력 | 기대감 낮음 ("무료니까 해볼까") | synco 앱 | 가입 과정 길면 이탈 |
| **활성화** | 엑셀 임포트 → 고객 50명+ 등록 | 놀라움 ("이게 되네") | 엑셀 업로드 | 엑셀 포맷 안 맞으면 좌절 |
| **Aha Moment** | AI가 고객 기업 뉴스 + 미팅 브리핑 제공 | 감탄 ("이건 쓸만하다") | AI 미팅 준비 | AI 품질 낮으면 바로 이탈 |
| **CEO 전환** | "CEO 가입시키면 수익 생긴다" 안내 → CEO에게 링크 발송 | 관심 ("이거 진짜?") | 가입 초대, 수익 안내 | 수익이 실감 안 나면 행동 안 함 |
| **수익 발생** | 자기 CEO 고객이 크레딧 구매 → 쉐어링 수익 입금 | 확신 ("이건 진짜다!") | 수익 대시보드 | 첫 수익까지 너무 오래 걸리면 이탈 |
| **확산** | 다른 FC에게 자발적 추천 | 자부심 ("나 이거 하는데") | 카톡, 대면 | — |

**핵심 Aha Moment:** AI 미팅 브리핑이 실제 미팅에 도움됨 (FC) + 첫 수익 쉐어링 입금 (FC)

## Journey 2: CEO (Demand Side)

```
FC 소개 → 가입 → 프로필 → 무료 CRM 활성화 → 매칭 조회 → 과금 → 연결 → 재사용
```

| Stage | 행동 | 감정 | 터치포인트 | 드롭오프 리스크 |
|-------|------|------|-----------|----------------|
| **FC 소개** | 담당 FC가 대면/카톡으로 synco(CEO 모드) 소개 | 신뢰 ("내 담당자가 추천하니 괜찮겠지") | FC 대면 | FC 설명이 부족하면 무관심 |
| **가입** | 링크 클릭, synco(CEO 모드) 기본 정보 입력 (무료) | 부담 없음 ("무료니까") | synco 앱 | 가입 5분 이상이면 이탈 |
| **프로필** | 사업 분야, 자원, 관심 분야 입력 | 약간의 귀찮음 | 프로필 입력 UI | **여기가 최대 드롭오프** — 정보 입력 동기 부족 |
| **무료 CRM 활성화** | 고객/인맥 등록, 기본 CRM 사용 시작 | 편리함 ("이건 쓸만하네") | CRM 대시보드 | CRM 가치 미인식 시 이탈 |
| **보너스 크레딧** | 프로필 80%+ 시 5 크레딧 지급 | 보상감 ("공짜 크레딧 받았네") | 크레딧 알림 | 크레딧 가치 미인식 |
| **매칭 조회** | AI 매칭 피드에서 블러 처리된 기회 확인 | 호기심 ("이게 뭐지?") | 매칭 피드 | 매칭 0건이면 즉시 이탈 |
| **Aha Moment** | 블러 해제 → "내 유휴 설비에 맞는 위탁 생산 수요" 확인 | 흥분 ("이런 게 있었어?!") | 매칭 상세 | **매칭 품질이 가치의 전부** |
| **과금** | 기업명 공개(2cr), 연결 요청(5cr) → 크레딧 추가 구매 | 합리적 판단 ("이 딜이면 5천원 아깝지 않다") | 크레딧 구매 | 가격 저항, 결제 UX |
| **연결** | 상대 CEO 수락 → 연락처 교환 | 만족 ("진짜 사업 파트너를 만났다") | 연결 성사 알림 | 상대가 수락 안 하면 불만 |
| **재사용** | 정기적으로 새 매칭 확인, 프로필 업데이트 | 습관화 ("사업 기회는 여기서 찾는다") | 푸시 알림 | 매칭 품질 저하 시 이탈 |

**핵심 Aha Moment:** 블러 해제 후 "내 사업에 실제로 맞는 기회"를 발견하는 순간

## Critical Drop-off Points & Mitigation

| 지점 | 리스크 | 확률 | 완화 전략 |
|------|--------|------|----------|
| **CEO 프로필 작성** | 정보 입력 귀찮음 | High | 보너스 크레딧(5cr) + "5분이면 끝" 가이드 + 단계적 입력 |
| **첫 매칭 품질** | 데이터 부족 → 매칭 0건/저품질 | High | 초기 수동 큐레이션 보완 + "데이터 쌓이면 더 좋아짐" 안내 |
| **크레딧 과금 전환** | "무료로 충분해" | Medium | 무료 3건으로 가치 경험 → "더 보려면 크레딧" 자연스러운 진행 |
| **FC의 CEO 전환 행동** | "귀찮아서 안 함" | Medium | 수익 쉐어링 실시간 표시 + CEO 가입 시 즉시 알림 |

## Flags

**Red Flags:** None

**Yellow Flags:**
- CEO 프로필 입력이 Cold Start의 핵심 병목 — 프로필 완성율 40% 미달 시 매칭 품질 전체가 무너짐
- FC→CEO 전환 플로우에서 FC의 "대면 소개 스크립트"가 없으면 전환율 급감 — 마케팅 자료 별도 필요
