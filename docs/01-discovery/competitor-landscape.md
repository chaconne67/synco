**Confidence:** Medium

---

## Competitive Overview

한국에서 **FC 대면 채널 + AI 매칭 + CEO 무료 CRM + 딜 중개 수수료**를 결합한 경쟁자는 **전무**하다. synco의 진짜 경쟁자는 "지인 소개로 충분하다"는 CEO의 관성과, 리멤버(250만 회원, 684억 매출)의 B2B 매칭 진출이다.

**시장 집중도:** 극도로 분산 (Fragmented) — GA CRM 지배적 플레이어 없음
**전체 위협 수준:** Medium — 직접 매칭 경쟁자 부재, 리멤버와 GA 자체 시스템이 간접 위협

## Competitor GTM Summary

### 리멤버 (잠재적 최대 위협)
- **GTM:** 명함 스캔 무료 → B2B SNS → 유료 프리미엄 (Black/Pro)
- **강점:** 250만 회원, 684억 매출, EQT 약 5,400억원 인수 자금력, 2026.3 매칭 런칭
- **약점:** FC 대면 채널 없음, 보험 도메인 전문성 없음, CEO 직통 전화번호 DB 없음

### SOLUTUS (GA CRM 유의미한 경쟁자)
- **GTM:** 보험사 API 계약 → 대형 GA MOU → 일괄 도입 (Top-Down)
- **강점:** 8개 보험사 레퍼런스, 인카 1,200명 협약
- **약점:** 8명 팀, Pre-A, 비즈매칭 방향성 없음

### 대형 GA 자체
- 굿리치: 자체 플래너 앱 + 오프라인 라운지 24개 + AI CRM
- 인카: 차세대 시스템 + SOLUTUS 도입
- **시사점:** 대형 GA는 자체 구축 중 → 중소형 GA가 synco의 기회

## Strategic Recommendations

### 어디서 경쟁할 것인가
1. **중소형 GA (4,360개, ~39,000명)** — 대형 GA가 자체 구축하는 동안 언더서브드 선점
2. **CEO 비즈니스 매칭** — 어떤 경쟁자도 FC 대면 채널 + AI 매칭을 결합하지 못함
3. **무료 CRM 트로이 목마** — 리멤버와 다른 진입 전략. FC/CEO 양쪽에 무료 제공

### 어디를 피할 것인가
1. **대형 GA 본사 직접 영업** — SOLUTUS + 자체 개발이 진행 중
2. **리멤버와 정면 경쟁** — B2B SNS/명함 영역에서 싸울 필요 없음
3. **보험 비교/추천** — 레드오션

### 차별화 공식
```
synco = 무료 CRM (트로이 목마) + AI 비즈니스 매칭 + FC 딜 중개
리멤버 = 명함 관리 → B2B SNS → 온라인 매칭
BNI = 대면 네트워킹 → 오프라인 소개
SOLUTUS = AI 보험분석 → 대형 GA Top-Down
```

## Flags

**Red Flags:**
- 리멤버 2026.3 매칭 런칭 + EQT 약 5,400억원 자금력 — 속도전에서 밀릴 수 있음

**Yellow Flags:**
- SOLUTUS 인카 1,200명 협약 — 대형 GA 선점 가속 중
- 대형 GA 자체 디지털 생태계 구축 → 장기적 외부 SaaS 기회 축소
- INFORYOU 약 182억원 투자 미검증 — 숨은 강자 가능성
