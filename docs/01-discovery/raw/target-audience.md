# 타겟 오디언스 프로파일링: GA 설계사 + 법인 대표 CEO

## 1. Primary Persona: GA 보험설계사

### 1.1 인구통계 프로필

| 항목 | 내용 |
|------|------|
| **연령대** | 핵심 타겟 40~55세 (생보 설계사 평균 51.4세, 신규 유입 20~30대 27.7%) |
| **성별** | 여성 비율 약 60~65% (생보 기준), 손보는 남성 비율 상대적으로 높음 |
| **총 시장 규모** | GA 소속 설계사 약 28.5만명 (2024), 전속 포함 전체 모집인 약 47만명 |
| **학력** | 전문대졸~대졸이 주류, 무경력 신입 31.1% (2024 GA 신규 설계사 기준) |
| **소득 구조** | 100% 수수료 기반, 상위 20%가 전체 수익의 60% 이상 차지 |
| **생산성** | 50대 월 신계약 45만원(최고), 40대 30만원, 60대 25만원, 20대 10만원 |

### 1.2 업무 행태 및 일과

| 시간대 | 활동 |
|--------|------|
| **08:00~09:30** | 출근, 조회(GA 지점 모닝 미팅), 보험사 공지 확인 |
| **09:30~12:00** | 기존 고객 콜/문자, 신규 미팅 일정 잡기, 설계서 작성 |
| **12:00~13:00** | 점심 (고객 또는 동료와 식사 겸 네트워킹) |
| **13:00~18:00** | 외근 — 고객 방문 상담, 계약 체결, 클레임 서포트 |
| **18:00~20:00** | 귀점 후 실적 정리, 설계서 재작업, 다음날 스케줄 수립 |
| **수시** | 카카오톡으로 고객 응대, 보험료 비교 견적 작성 |

### 1.3 Pain Points (핵심 고충)

1. **고객 관리 분산**: 엑셀, 카카오톡, 수첩, 보험사별 영업지원시스템(최소 5~10개 보험사 시스템 별도 사용)에 데이터 분산
2. **법인 영업 진입장벽**: CEO 고객 접점은 풍부하나(1인당 1,000~3,000명), 법인 보험/비보험 니즈를 체계적으로 파악할 도구 부재
3. **반복 행정 과부하**: 설계서 비교, 수수료 정산 확인, 계약 관리 등 비영업 업무가 일과의 40% 이상
4. **정보 비대칭**: 보험사별 상품 변경, 요율 변경 정보를 실시간 추적하기 어려움
5. **이직 시 고객 데이터 유실**: GA 간 이동 빈번하나 고객 DB는 소속 GA 시스템에 종속
6. **디지털 역량 격차**: 50대 이상 핵심 생산층이 새 IT 도구 학습에 저항감

### 1.4 기술 사용 현황

- **필수 도구**: 카카오톡(고객 소통 1순위), 각 보험사 영업지원 앱(삼성 iSP, 한화 LIFE Plus 등), 엑셀/구글시트
- **GA 본사 제공 시스템**: 수수료 정산 시스템, 계약 조회 시스템 (대부분 웹 기반, UX 열악)
- **개인 활용**: 네이버 카페/블로그(고객 유입), 인스타그램(브랜딩), 유튜브(학습)
- **CRM 사용률**: 별도 CRM 도구 사용 비율 매우 낮음 (추정 5% 미만), 대부분 엑셀 또는 카카오톡 채팅방으로 대체
- **인슈어테크 플랫폼**: 라이프리(솔루투스), H.AI(라이프시맨틱스), 토스인슈어런스 디지털 온보딩 등 최근 등장

### 1.5 구매 의사결정 특성

| 요소 | 설명 |
|------|------|
| **의사결정 주체** | GA 본사 IT팀(Top-down) vs. 개인 설계사(Bottom-up) 이중 구조 |
| **Top-down 경로** | GA 대표/IT 담당 → 벤더 데모 → 파일럿 지점 테스트 → 전사 도입 (6~12개월) |
| **Bottom-up 경로** | 개인 설계사가 무료 도구 시작 → 지점 내 확산 → GA 본사 공식 도입 검토 |
| **가격 민감도** | 매우 높음 — 수수료 수입에서 직접 지출, 월 1~3만원 수준이 심리적 한계 |
| **전환 비용 인식** | 기존 엑셀/수첩 데이터 마이그레이션 부담, "배우는 시간 = 영업 못하는 시간" |
| **신뢰 기반** | 동료 설계사 추천, 지점장 권유가 결정적 — 광고보다 입소문 |
| **핵심 판단 기준** | "이걸 쓰면 계약이 더 나오는가?" (ROI 즉시 체감 필요) |

## 2. Secondary Persona: 중소기업 대표 (CEO)

### 2.1 인구통계 프로필

| 항목 | 내용 |
|------|------|
| **연령대** | 45~65세 (중소기업 대표 평균 연령 55세 전후) |
| **성별** | 남성 80% 이상 |
| **기업 규모** | 직원 5~100명, 연매출 10억~500억원 |
| **업종** | 제조업, 건설업, 도소매업, 서비스업 순 |
| **보험 관계** | 법인 보험(단체보험, 배상책임, 재물, 퇴직연금) + 대표 개인 보험 |

### 2.2 보험 관련 니즈

1. **비용 절감**: 법인 보험료 최적화, 중복 보장 제거
2. **리스크 관리**: 배상책임, 산재, 핵심인력 보장 등 경영 리스크 대비
3. **세제 혜택**: 법인 보험을 통한 절세 구조 설계
4. **퇴직연금**: DC/DB 전환 비교, 수수료 최적화
5. **비보험 니즈**: 세무/법무 자문, 사업 확장을 위한 네트워킹, 자금 조달 정보

### 2.3 디지털 도구 채택 행동

| 특성 | 설명 |
|------|------|
| **디지털 리터러시** | 보통~낮음 (카카오톡, 네이버 검색이 주요 디지털 접점) |
| **정보 습득 채널** | 지인 추천 > 네이버 카페/블로그 > 유튜브 > 전문 매체 |
| **의사결정 스타일** | 신뢰하는 사람(설계사, 세무사, 동종업계 대표) 추천에 크게 의존 |
| **앱 사용 의향** | 직접 앱 설치/사용보다 카카오톡 링크, 웹페이지 형태 선호 |
| **시간 가용성** | 매우 제한적 — 15분 이내 핵심 정보 전달이 관건 |

### 2.4 Pain Points

1. **보험 포트폴리오 파악 불가**: 본인/법인 명의 보험이 몇 개인지, 보장 내용이 무엇인지 모름
2. **갱신 시기 관리 실패**: 자동차보험, 배상책임보험 등 갱신 시점을 놓쳐 공백 발생
3. **비교 정보 부족**: 여러 설계사가 각자 유리한 상품만 추천, 객관적 비교 불가
4. **비보험 경영 정보 갈증**: 세무, 법무, 보조금, 네트워킹 등 원스톱 창구 부재

## 3. Anti-Persona (비타겟)

### 3.1 제외 대상: 설계사 측

| 유형 | 제외 사유 |
|------|-----------|
| **1년 미만 신입 설계사** | 고객 DB 미형성, 이탈률 극히 높음(1년 내 50% 이상 이탈), 유료 도구 지불 의향 없음 |
| **보험사 전속 설계사** | 단일 보험사 상품만 취급, 비교 추천/비즈니스 매칭 니즈 낮음, 소속사 시스템 사용 의무 |
| **온라인 전용 채널 설계사** | TM/CM 채널, 대면 영업 없이 디지털 퍼널 운영 — CRM 니즈 상이 |
| **소형 1인 GA** | 고객 수 100명 미만, 비용 지불 여력 부족, ROI 실현 어려움 |

### 3.2 제외 대상: CEO 측

| 유형 | 제외 사유 |
|------|-----------|
| **대기업/상장사 대표** | 전담 보험 브로커/리스크관리팀 보유, GA 설계사 채널 불필요 |
| **1인 자영업자** | 법인 보험 니즈 미미, 비즈니스 매칭 가치 낮음 |
| **스타트업 초기(Pre-revenue)** | 보험 우선순위 극히 낮음, 지불 여력 부족 |

## 4. 구매 행동 모델

### 4.1 GA 본사(Top-down) 도입 경로

```
인지 → IT/경영진 데모 요청 → 파일럿 지점(2~3개) 운영 → 효과 검증(3~6개월) → 전사 롤아웃
```

- **의사결정자**: GA 대표, CTO/IT 담당 임원
- **영향자**: 지점장, 스타 설계사(High Performer)
- **결정 기준**: (1) 기존 시스템 연동 가능성, (2) 설계사 이탈 방지 효과, (3) 수수료 정산 정확도
- **예산 구조**: GA 본사 IT 예산 (설계사당 월 5,000~20,000원 수준)
- **계약 형태**: SaaS 구독형, 연간 계약 선호

### 4.2 개인 설계사(Bottom-up) 도입 경로

```
동료 추천 / SNS 발견 → 무료 체험 → 3건 이상 계약 연결 체감 → 유료 전환 → 지점 내 전파
```

- **전환 트리거**: "이 도구 덕분에 계약 나왔다"는 성공 체험 1회
- **이탈 트리거**: 3주 내 효과 미체감 시 삭제
- **지불 방식**: 카드 자동결제, 카카오페이/네이버페이 선호
- **가격 수용 범위**: 무료~월 29,000원 (프리미엄 기능)

### 4.3 핵심 의사결정 기준 (Decision Criteria)

| 순위 | 기준 | 가중치 |
|------|------|--------|
| 1 | **영업 성과 직결성** — 실제 계약 증가에 기여하는가 | 35% |
| 2 | **사용 편의성** — 카카오톡 수준의 직관적 UX | 25% |
| 3 | **데이터 이식성** — 엑셀 임포트, GA 이동 시 데이터 휴대 가능 | 15% |
| 4 | **가격 합리성** — 월 수수료 대비 ROI 체감 | 15% |
| 5 | **보안/신뢰** — 고객 개인정보 보호, 금융당국 규제 준수 | 10% |

## 6. 핵심 인사이트 요약

1. **GA 설계사는 '도구'가 아니라 '계약 결과'를 산다** — 기능 나열 마케팅 무효, 성공 사례 기반 커뮤니케이션 필수
2. **카카오톡이 사실상의 CRM이다** — 카카오톡 연동/공유 기능이 없으면 채택 불가
3. **지점장이 게이트키퍼** — B2B 세일즈에서 지점장 확보가 가장 효율적인 GTM
4. **50대 핵심 생산층의 UX 장벽** — 최고 생산성 연령대가 디지털 도구에 가장 저항적, 극도의 단순 UX 필요
5. **CEO는 직접 사용자가 아니다** — CEO에게는 설계사가 전달하는 '가치 콘텐츠'로 접근, 앱 설치 요구 금지
6. **GA 본사 IT 도입은 느리지만 확실하다** — SaaS 구독형, 파일럿 → 전사 확산 모델, 기존 시스템 연동 필수
7. **데이터 휴대성이 잠금(Lock-in) 해제 열쇠** — GA 간 이동이 잦은 설계사에게 "내 데이터는 내 것" 메시지가 강력한 차별화
