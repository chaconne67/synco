## Pivot: 양면 플랫폼 + CEO 과금 모델 (2026-03-25)

### 배경
- 기존 모델: 설계사 SaaS 과금 (Pro 월 3.9만원) + 매칭 수수료
- 문제: 설계사 WTP 낮음 (하위 60%가 0~2만원), SaaS 천장 낮음

### 새 수익 모델 핵심
- **설계사(FC) = 무료.** AI 비서를 무상 제공 → 적극적 참여 유도
- **CEO = 과금 대상.** FC를 통해 기 확보된 CEO를 회원 가입 (가입 무료)
- **3단계 플랜 과금 차별화** + **부분유료화** (관심 보인 정보에 과금, 게임 모델)
- **수익 쉐어링:** CEO 과금 수익을 FC와 분배 → FC가 CEO를 끌어오는 인센티브
- **딜 중개 수수료** 3-10% (매치메이커 모델, 가장 큰 시장 500-2,000억)
- **광고 수입** 가능 (CEO 타겟 B2B 광고, 금융사 유튜브 2천만+)
- **금융상품 유통** (Layer 3): CEO DB 10만+ 시 대면 유통 채널로 전환

### 핵심 테제: DB가 플랫폼이다 (세션 3)
- CEO 직통 전화번호 + FC 대면 채널 = 돈으로 복제 불가한 자산
- GA에게는 추가 가치 0인 유휴 자산 → synco가 활성화
- 활성화된 CEO DB 위에 무한한 파생 비즈니스 가능
- AI 시대 관심은 크지만 실제 도입은 극소수 → 무료 CRM으로 진입 최적 타이밍

### 양면 플랫폼 구조
```
Supply (무료): FC → CEO 데이터 + 대면 관계 제공
Platform (AI): 무료 CRM → 유휴 자원 분석 → 매칭 기회 발굴
Demand (유료): CEO → 무료 CRM 진입 → 유료 매칭 + 딜 중개
Revenue: CEO 구독/크레딧 + 딜 수수료 3-10% + 광고 + Layer 3
```

### 3-Layer 전략
- Layer 1: 무료 CRM + AI 비서 (FC 락인, CEO 가입 유도) — 트로이 목마
- Layer 2: CEO 과금 (3단계 플랜 + 부분유료화 + 딜 중개 수수료 + 광고)
- Layer 3: 금융상품 대면 유통 채널 (로보어드바이저, 투자자문 등)
