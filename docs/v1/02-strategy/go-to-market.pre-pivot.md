## Launch Strategy

### CRM-First, 가볍고 빠르게

```
Month 1-2:  무료 CRM MVP → 공동창업자 GA 30명 파일럿
Month 3-4:  성공 케이스 확보 → AI Pro 기능 추가
Month 5-6:  인접 GA 2~3개 확장 → Enterprise 가격 모델 테스트
Month 7-12: 키맨 설계사 네트워크 → 20개 GA 목표
```

> 핵심 원칙: **대면 영업자가 키맨이 되면 모든 게 해결된다** (창업자 방향)

## 키맨 전략 상세

### 키맨이란?
- GA 내에서 디지털 도구를 적극 사용하고, 성과를 입으로 전파하는 설계사
- 보통 팀장급, 40대 중반, 동료에게 영향력 있는 사람
- 1개 GA에 1~2명만 확보하면 해당 GA 전체로 확산

### 키맨 확보 전략
```
1. 첫 GA에서 가장 적극적인 사용자 3명 식별
2. 1:1 밀착 지원 — 데이터 임포트, AI 설정, 맞춤 교육
3. 성공 경험 만들기 — AI 추천으로 실제 계약/미팅 성사
4. 성공 스토리를 본인 입으로 말하게 함 (강요 아닌 자연스러운 자랑)
5. 키맨이 다른 GA 지점장에게 소개 → 연쇄 반응
```

### 키맨 인센티브
- Pro 기능 무료 제공 (3~6개월)
- 신규 사용자 추천 시 무료 기간 연장
- "베타 테스터" 뱃지 — 소셜 인정 욕구 충족
