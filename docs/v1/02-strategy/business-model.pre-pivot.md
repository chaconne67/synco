# Business Model: GA 비즈니스 매칭 플랫폼

**Phase:** Phase 4 — Strategy
**Project:** ga-biz-match
**Date:** 2026-03-25
**Confidence:** Medium

---

## Revenue Model

### 수익 구조: Freemium SaaS + Transaction Fee (단계적)

```
Phase 1 (0~6개월): Free CRM → 사용자 확보
Phase 2 (6~12개월): Pro AI 유료 전환 + Enterprise GA 계약
Phase 3 (12개월+): 비즈매칭 수수료 추가
```

### Pricing Tiers

| Tier | 기능 | 가격 | 전환 동기 |
|------|------|------|----------|
| **Free** | 고객 등록/메모/일정/알림, 엑셀 임포트, 기본 검색 | 0원 | 카톡+엑셀 대체, 데이터 안전 |
| **Pro** | AI 분석(재접촉 타이밍, 크로스셀), 가치 콘텐츠 생성, 고급 필터 | 월 3.9만원/user | "AI 덕분에 계약 나왔다" 경험 |
| **Enterprise** | Pro 전체 + 관리자 대시보드 + 지점 통계 + 우선 지원 | 설계사당 월 1.5~2만원 (100명+) | GA 본사 일괄 도입 |
| **매칭 수수료** | CEO 간 비즈니스 연결 성사 | 건당 3~5% | Phase 2, 데이터 축적 후 |

> 가격 근거: WTP 심리적 한계 월 2.9만원 [Data], Enterprise는 GA 본사 B2B 계약이 핵심 매출원 [Data, `target-audience.md`]

## Unit Economics

| 지표 | 추정 | 근거 | Confidence |
|------|------|------|-----------|
| **ARPU** | 월 3만원 (혼합) | Enterprise 70% × 2만원 + Pro 30% × 3.9만원 | Medium |
| **CAC** | 10~20만원 | 공동창업자 네트워크 활용, 유료 광고 없음 [Estimate] | Medium |
| **LTV** | 72~108만원 | ARPU 3만 × (1/월 이탈률 4%) = 24개월 | Low |
| **LTV:CAC** | 4:1~7:1 | 네트워크 기반 저CAC 전략 | Low |
| **월간 이탈률** | 3~5% (초기) | 한국 SMB SaaS 벤치마크 [Data, `market-analysis.md`] | Medium |
| **Payback Period** | 3~7개월 | CAC 10~20만 ÷ ARPU 3만 | Low |

> CAC가 벤치마크(30~50만원)보다 낮은 이유: 공동창업자 GA 네트워크 + 키맨(설계사) 입소문 채널 [Opinion]

## Break-Even Analysis

```
월 고정비: 85만원 (Phase 2 기준)
ARPU: 3만원
변동비/user: 2,000원
공헌이익/user: 28,000원

BEP = 850,000 ÷ 28,000 ≈ 31명 유료 사용자
```

> **31명 유료 사용자로 월 손익분기** — 첫 GA 30명 중 유료 전환 목표와 정확히 일치. 현실적으로 달성 가능. [Estimate]

## Scalability

| 단계 | 확장 경로 | 한계 |
|------|----------|------|
| 0~500명 | 공동창업자 네트워크 → 인접 GA | 창업자 시간 제약 (사이드 프로젝트) |
| 500~3,000명 | 키맨 설계사 입소문 + SEO | 콘텐츠 제작 역량 |
| 3,000~10,000명 | Enterprise GA 계약 + 매칭 네트워크 효과 | 컴플라이언스 비용 증가 |
| 10,000명+ | 비즈매칭 플랫폼 효과 | 규제 대응, 조직 확대 필요 |

## Key Dependencies

1. **공동창업자 GA 네트워크** — 첫 100명 확보의 유일한 채널
2. **AI API 비용 안정성** — OpenAI/Anthropic 가격 변동에 수익성 영향
3. **개인정보 위탁 처리 구조** — 법률 자문으로 확정 필요
4. **키맨 설계사 확보** — 1~2명의 열성 사용자가 지점 전체 확산 결정

## Flags

**Red Flags:** None

**Yellow Flags:**
- Year 1 매출 3.6억원은 2인 사이드 프로젝트로는 충분하나, 풀타임 전환이나 투자 유치에는 부족할 수 있음
- AI API 비용이 사용자 증가와 비례 — 고사용 유저 발생 시 마진 압박
- 매칭 수수료(Phase 3) 매출 규모는 완전히 미검증 [Assumption]
