## 1. 업종별 중개 수수료율 총괄

### 1.1 M&A 중개

| 구분 | 수수료율 | 비고 |
|------|---------|------|
| **리먼 공식 (Lehman Scale)** | 1st $1M: 5%, 2nd: 4%, 3rd: 3%, 4th: 2%, 이후: 1% | 전통적 M&A 자문 수수료 공식 |
| **더블 리먼 (Double Lehman)** | 1st $1M: 10%, 2nd: 8%, 3rd: 6%, 4th: 4%, 이후: 2% | 현대 비즈니스 브로커 실무 적용 |
| **한국 M&A 중개** | 거래금액의 1~10% (성공보수 기준) | 법정 기준 없음, 협의 결정 |
| **한국 M&A 자문** | 착수금 + 성공보수 (거래액 2~5%) | 대형 회계/법률법인 기준 |

- 한국은 M&A 중개 수수료에 법정 요율 없음. 정보제공료, 착수계약금, 성공수수료, 자문료 등으로 구성 [Data]
- 2026년 중소벤처기업부: M&A 가치평가 수수료 최대 60%(2,000만원 한도) 지원사업 시행 [Data]

> 출처: [한국M&A거래소](https://kmx.kr/bbs/content.php?co_id=commission2), [Rejigg](https://www.rejigg.com/articles/business-broker-commission-rates), [한국금융신문](https://www.fntimes.com/html/view.php?ud=2002012319375751984_18)

### 1.2 비즈니스 브로커 수수료 (미국/글로벌)

| 거래 규모 | 수수료율 | 비고 |
|-----------|---------|------|
| $1M 미만 (소규모) | **8~12%** | 최소 수수료 $10,000~$50,000 적용 |
| $1M~$5M (중소규모) | **5~10%** | 더블 리먼 적용 다수 |
| $5M~$25M (중규모) | **5~8%** (blended) | 리테이너 $5,000~$25,000 별도 |
| $25M 이상 (대규모) | **1~4%** | 리테이너 $50,000+ |

- 리테이너: $5,000~$25,000 (비환불), 월 마케팅비: $500~$2,500/월 [Data]
- 총비용(수수료+부대비용) 합산 시 매각가의 최대 15%까지 발생 가능 [Data]
- $3M 매각 시 실제 총비용 약 $259,000 (8.6%) 예시 [Data]

> 출처: [Rejigg](https://www.rejigg.com/articles/business-broker-commission-rates), [ClearlyAcquired](https://www.clearlyacquired.com/blog/how-much-do-business-broker-fees-actually-cost-in-2026), [MidStreet](https://www.midstreet.com/blog/business-broker-fees-when-selling-a-business)

## 4. B2B 매칭 플랫폼 수수료 설계 시사점

### 4.1 핵심 인사이트

1. **거래 규모 반비례 법칙**: 모든 업종에서 거래 금액이 커질수록 수수료율은 낮아짐 (리먼 공식이 대표적)
2. **초기에는 한측 과금이 유리**: 수요-공급 중 한쪽을 무료로 해서 유저 확보 후, 점진적으로 양측 과금 전환 (위시켓, 리멤버 사례)
3. **멤버십 vs 거래수수료 선택**: 대형 B2B(Alibaba)는 멤버십, 소규모 거래 중심은 거래수수료가 일반적
4. **최소 수수료 설정 필수**: 소규모 거래에서 운영비 커버를 위해 필수 (비즈니스 브로커 사례)
5. **성공 보수 모델의 신뢰성**: 성공 시에만 과금하는 모델이 중소기업 CEO 초기 신뢰 확보에 유리

### 4.2 한국 중소기업 B2B 매칭 플랫폼 권장 수수료 구조 [Assumption]

| 거래 규모 | 권장 수수료율 | 수수료 구조 |
|-----------|-------------|------------|
| 100만~1,000만원 | 10~15% (또는 최소 10만원) | 성공 보수형, 한측(공급자) 과금 |
| 1,000만~5,000만원 | 5~10% | 성공 보수형, 양측 과금 가능 |
| 5,000만~1억원 | 3~7% | 하이브리드 (기본료 + 성과보수) |
| 1억원 이상 | 2~5% | 리테이너 + 성공 보수 |

### 4.3 수수료 외 수익원 고려

- 프리미엄 리스팅/노출 광고 (Alibaba 모델)
- 데이터/인사이트 구독 서비스
- 에스크로/결제 수수료 (2~3%)
- 부가 서비스 (법률 검토, 계약서 작성, 실사 지원)
