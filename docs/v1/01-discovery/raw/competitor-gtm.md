- **Phase**: Phase 3 — 경쟁사 GTM 리서치
- **Project**: ga-biz-match
- **Date**: 2026-03-25
- **Sources**: 웹 검색 10회 + 페이지 분석 6건

## 1. 솔루투스(SOLUTUS) — GTM Breakdown

### 기업 개요
- **설립**: 2019년 3월, 서울 서초구
- **대표**: 강승우
- **임직원**: 8명 (2026년 2월 기준)
- **투자 라운드**: Pre-A (2025년 8월 전략적 투자 유치, 금액 비공개)
- **핵심 제품**: 라이프리(Lifree) — 예상보험금 기반 보험분석 플랫폼
- **기술**: 빅데이터·AI 기반 예상보험금 산출, XAI 적용

### Primary Acquisition Channel: B2B 전략적 파트너십 (보험사 → GA 확장)
- **1단계**: 8개 대형 보험사와 API 솔루션 공급 계약 체결 → 기술 검증
- **2단계**: 검증된 기술을 GA 시장으로 확장 (인카금융서비스 J-Us 사업단 1,200명 설계사 대상 협약)
- **핵심 전략**: "보험사에서 먼저 검증 → GA로 레퍼런스 확장"이라는 Top-Down 모델

### Sales Motion: Enterprise Sales-Assisted
- 보험사 대상 API 솔루션 B2B 직접 영업
- GA 대상은 전략적 협약(MOU) 체결 후 일괄 도입
- 개별 설계사 대상 셀프서브 아님 — GA 법인 단위 도입

### Content Strategy
- 보도자료 중심의 PR (청년개발자신문, 뉴스와이어 등)
- 별도 블로그/콘텐츠 마케팅 미확인
- 앱스토어(Google Play) 통한 B2C 노출 (라이프리 소비자용 앱)

### Partnership Plays
- 인카금융서비스(대형 GA, 코스닥 상장) 전략적 협약
- 8개 보험사 API 파트너십
- DB손해보험 전략적 투자 연계

### 약점/기회
- 팀 규모 8명으로 매우 소규모 → 확장 속도 제한
- Pre-A 단계로 자금력 제한적
- B2C 앱(라이프리 소비자용)은 별도 존재하나 마케팅 집중도 낮음

## 2. 인포유(INFORYOU) — GTM Breakdown

### 기업 개요
- **위치**: 광주 광산구 (JobKorea 기준)
- **투자**: $13.5M 유치 정보 (사용자 제공, 웹 검색으로 상세 확인 불가)
- **서비스**: 인포유금융(infis.co.kr) — 보험 관련 금융 서비스
- **채널**: 카카오톡 채널 운영 중

### Primary Acquisition Channel: 확인 제한적
- 공개된 GTM 정보가 극히 제한적
- 카카오톡 채널을 운영하고 있어 모바일 메신저 기반 고객 접점 활용 추정
- 광주 소재로 지역 기반 영업 가능성

### Sales Motion: 정보 부족으로 추정 불가
- 웹사이트(infis.co.kr) 존재하나 상세 서비스 모델 공개 정보 제한

### Data Gap
- 인포유의 구체적 서비스, 타겟 고객, 영업 방식에 대한 공개 정보가 매우 부족
- $13.5M 투자 이력도 웹 검색으로 교차 검증 실패
- 추가 조사 필요: 직접 사이트 방문, 업계 인터뷰 등

## 3. 간접 경쟁사 — 대형 GA 자체 솔루션

### 굿리치(GoodRich)
- **포지션**: GA이자 보험 플랫폼 기업
- **GTM**: 소비자 앱(보험 조회/보장분석/보험금 청구) + 설계사용 '굿리치플래너'(2020년 출시)
- **오프라인**: 굿리치라운지 전국 24개 점포 → 2025년 추가 확장
- **전략**: IDC → 클라우드 이관, 디지털 생태계 자체 구축
- **시사점**: 대형 GA는 자체 IT 솔루션을 구축하는 추세 → 외부 SaaS 도입 필요성 낮음

### 인카금융서비스
- 2023년 인프라 도입, 2024년 차세대 시스템 개발 착수
- 솔루투스 라이프리 2.0 도입 (J-Us 사업단 1,200명)
- **시사점**: 대형 GA도 전문 인슈어테크 솔루션 외부 도입 사례 존재

### 뱅크샐러드금융서비스
- 데이터 기반 디지털 보험 상담 GA
- 고객 금융·건강 마이데이터 분석 → AI 기반 표준화 상담 체계
- **시사점**: 데이터 활용이 GA의 핵심 경쟁력으로 부상

### 해빗팩토리
- AI 기반 비대면 보험상품 비교·상담
- 시리즈D 350억원 투자 유치
- **시사점**: 보험 비교/추천 영역은 이미 대형 투자 유치 완료, 레드오션

## 4. 한국 B2B SaaS 시장 영업 모델 분석

### PLG vs SLG — 한국 시장 현실

| 구분 | PLG (Product-Led Growth) | SLG (Sales-Led Growth) |
|------|--------------------------|------------------------|
| 한국 적용성 | 낮음 — 자발적 전파 구조 부족 | 높음 — 레퍼런스 중심 의사결정 |
| 성공 사례 | 한국 PLG 단독 성공 사례 거의 없음 | 대부분의 한국 B2B SaaS |
| 권장 전략 | **하이브리드**: PLG + 세일즈 조직 병행 | 어카운트 매니저 중심 |

출처: 카카오벤처스 블로그

### 한국 B2B SaaS 핵심 인사이트
1. **"어디에서 쓰는 서비스인지"가 신뢰도를 좌우** → 초기 레퍼런스 GA 확보가 최우선
2. **스타트업 고객은 단가가 낮아 지속 성장 담보 어려움** → 중견 이상 GA 타겟팅 필요
3. **어카운트 매니저(챔피언)** 발굴이 핵심 — GA 내 의사결정자와의 관계 구축
4. **국내 SaaS 시장 규모**: 2020년 5,780억 → 2025년 1조 1,430억 (CAGR 14.9%)

## 6. GA 디지털전환 환경 분석

### GA 협회 추진 방향 (2026년)
- 슬로건: "소비자 권익 보호 강화를 통한 GA 지속가능한 경영환경 구축"
- **전산·보안 시스템 고도화** 핵심 과제로 채택
- 2030년 보험판매전문회사 전환 → 2035년 금융판매전문회사 도약 로드맵
- GA 교육 기능 강화 추진 중

### 시장 구조적 기회
- 301,275명 GA 설계사, 84%가 74개 대형 GA에 집중
- DB 리스크(해킹 사고) 후폭풍으로 영업모델 전환 압박 가중
- N잡 설계사 확산 — 영업 구조 변화로 디지털 도구 수요 증가
- 메리츠화재 플랫폼 기반 설계사 조직 4,200명 → 10,000명 돌파 사례

## 7. Key Takeaway

### 경쟁사에서 작동하는 전략
1. **솔루투스**: 보험사 레퍼런스로 기술력 검증 → GA 법인 대상 Top-Down 확장
2. **굿리치/인카**: 자체 디지털 생태계 구축으로 설계사 Lock-in
3. **해빗팩토리**: 대규모 VC 자금으로 B2C 보험 비교 시장 선점

### 미개척 채널 (Under-exploited)
1. **보험 설계사 특화 콘텐츠 SEO** — "보험 설계사 CRM", "보험 고객관리" 등 키워드에 전문 콘텐츠 부재
2. **중소형 GA 시장** — 경쟁사 모두 대형 GA/보험사에 집중, 중소형 GA는 방치 상태
3. **설계사 커뮤니티 마케팅** — 카카오톡 오픈채팅, 네이버 카페, 밴드 등 설계사 커뮤니티 공략 부재
4. **GA 협회 정책 연계** — 디지털전환 정책에 부합하는 솔루션으로 포지셔닝

### ZERO Budget 승리 전략

#### 즉시 실행 가능 (0원)
1. **공동창업자 GA 네트워크 = 유통 채널**: 첫 5~10개 GA에 무료 파일럿 제공 → 레퍼런스 확보
2. **설계사 커뮤니티 침투**: 보험 설계사 카페/밴드/오픈채팅에서 실질적 가치 제공 (영업 노하우, CRM 팁)
3. **콘텐츠 SEO**: "보험 설계사 고객관리", "GA CRM" 등 키워드 블로그 콘텐츠 선점
4. **Founder-Led Sales**: 창업자가 직접 GA 방문, 현장 데모 → 신뢰 구축

#### 초기 단계 (최소 비용)
5. **GA 내부 교육 세미나**: AI·디지털 도구 활용법 무료 교육 제공 → 자연스러운 제품 노출
6. **추천 프로그램**: 기존 사용 설계사가 동료 추천 시 인센티브 (구전 효과)
7. **GA 협회 디지털전환 과제와 연계**: 협회 행사/세미나 참여로 업계 인지도 구축

### 공동창업자 GA 네트워크 활용 로드맵

```
[Month 1-2] 네트워크 내 GA 5곳 무료 파일럿
     ↓
[Month 3-4] 파일럿 데이터로 사례 연구(Case Study) 작성
     ↓
[Month 5-6] 사례 연구 기반 콘텐츠 마케팅 + 인접 GA 확장
     ↓
[Month 7-12] 추천 프로그램 가동 + GA 협회 채널 활용
```

**핵심 원칙**: 한국 B2B 시장에서는 "어디에서 쓰는 서비스인지"가 신뢰도를 결정한다. 따라서 첫 레퍼런스 GA 확보가 모든 GTM의 출발점이다.
