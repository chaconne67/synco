## B2B 마이크로트랜잭션 성공/실패 조건

**작동하는 조건:**
1. 정보의 즉시 가치가 명확 (연락처 1건 → 딜 가능성)
2. ROI 계산 가능 (크레딧 1개 = 어떤 가치?)
3. 크레딧 소진 압박 적당 (월별 적립·소멸)
4. 무료 경험 선행 (데이터 품질 확인 후 과금)

**실패 조건:**
- 정보 품질 낮거나 예측 불가 → 신뢰 붕괴
- 단가 너무 높거나 낮음
- 크레딧 체계 복잡 → 계산 피로

## 3단계 플랜 설계 벤치마크

**가격 비율:** Free : Pro : Enterprise = 1 : 3~5 : 10~15
- 83% B2B SaaS가 Free Forever 티어 제공
- **59%가 사용량 제한**을 주 차별화 (기능 제한보다 효과적)
- 61%가 하이브리드 과금 (정액 + 사용량 + 유저 수)

**핵심 시사점:**
- "조회는 되지만 세부는 블러" → 기능 제한보다 전환율 높음
- 정보 레이어별 차등 과금 (Apollo 방식) → CEO 매칭에 적용 가능
- LinkedIn InMail 크레딧 환불 → 연결 성사 시 크레딧 반환으로 신뢰 구축
- 프로필 완성도 점수 → 데이터 입력 인센티브
