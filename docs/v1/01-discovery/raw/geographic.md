## 2. 지역별 분포 분석

### GA 지역 집중도 (추정)

공식 지역별 GA 분포 통계는 비공개이나, 다음 구조적 요인으로 추정:

- **수도권 (서울/경기/인천)**: 전체 GA 설계사의 **55-60% 추정**
  - 인구 비중 50%+, 보험 가입률/소득 수준 최고
  - 대형 GA 본사 대부분 서울 소재 (한화금융서비스, 인카, 에이플러스에셋 등)
  - 경기 남부 (성남/분당, 수원, 용인): 고소득 + 중소기업 밀집
- **부산/경남권**: 약 10-12%
- **대구/경북권**: 약 7-8%
- **대전/충청권**: 약 7-8%
- **광주/전라권**: 약 5-6%

### 중소기업 밀집 지역 (보험 수요 원천)

| 지역 | 특성 | 보험 수요 |
|------|------|-----------|
| 경기 남부 (화성/평택/안산) | 제조업 산업단지 집중, 삼성/현대 협력사 밀집 | 기업보험, 단체보험, 대표자/임원 보험 |
| 경기 북부 (파주/김포/고양) | 물류/유통 + 신도시 개발 | 화재/배상책임, 개인보험 |
| 서울 강남/서초 | 서비스업/IT/전문직 밀집 | 고액 종신, 변액, 법인보험 |
| 서울 영등포/구로/금천 | IT/스타트업 집중 (G밸리) | 기업보험, D&O, 키맨보험 |
| 인천 남동/서구 | 제조업 산업단지 | 기업보험, 단체보험 |

## 3. Beachhead 시장 추천

### 1순위: 수도권 중소형 GA (30-200명 규모) x 경기 남부 중소기업

**Why this beachhead:**

1. **첫 파트너 기반 활용**: ~30명 설계사, 50-100K DB를 가진 첫 GA 파트너가 특정 지역 기반 -> 해당 지역에서 실증(PoC) 후 확장
2. **중소형 GA의 Pain Point가 가장 큼**:
   - 대형 GA는 자체 CRM/시스템 보유 (한화: 자체 플랫폼, 인카: 보맵 연계)
   - 중소형 GA (4,214개, 설계사 47,830명)는 디지털 도구 부재
   - 설계사 이탈 방지가 최대 과제 -> CRM/비즈매칭이 직접적 가치 제공
3. **중소기업 밀집 지역 = 비즈매칭 수요 극대화**:
   - 경기 남부 산업단지: 제조업 사업주 -> 기업보험+개인보험 교차판매
   - 사업주 네트워크가 밀집되어 비즈매칭(사업주 간 연결) 자연 발생
4. **Co-founder GA 네트워크**: GA 업계 인맥 활용 -> 같은 권역 중소형 GA 확장 용이

### Beachhead 정의

| 항목 | 정의 |
|------|------|
| **Target GA** | 수도권 소재, 30-200명 규모 중소형 GA |
| **Target 지역** | 첫 파트너 소재 지역 -> 인접 경기 남부 확대 |
| **Target 고객** | 중소기업 사업주 (제조/서비스업, 연매출 5-100억) |
| **핵심 가치** | 설계사에게 "기존 DB에서 추가 기회 발굴" + "사업주 간 비즈매칭으로 관계 강화" |
| **TAM (중소형 GA)** | 4,214개 GA x 평균 11명 = ~47,800명 설계사 |
| **초기 SAM** | 수도권 중소형 GA ~2,300개 x 11명 = ~25,000명 |
| **첫해 SOM** | 첫 파트너 + 인접 5-10개 GA = 150-500명 설계사 |

## 4. 확장 경로 (Expansion Path)

```
Phase 1 (0-6개월): 첫 GA 파트너 PoC
  - 30명 설계사, 50-100K DB
  - AI CRM 실증 + 비즈매칭 1차 모델 검증
  - KPI: 설계사당 월 미팅 수, 계약전환율

Phase 2 (6-12개월): 인접 중소형 GA 확장
  - 같은 지역 5-10개 중소형 GA
  - Co-founder 네트워크 활용한 직접 영업
  - 500-2,000명 설계사 규모

Phase 3 (12-18개월): 대형 GA 산하 지점 진입
  - 74개 대형 GA의 지역 지점 단위 도입
  - 대형 GA 지점장이 중소형 GA 대비 의사결정 빠름
  - 지점별 50-200명 -> 수천명 단위 확장

Phase 4 (18-24개월): 전국 + 보험사 채널 확장
  - 대형 GA 본사 계약 -> 전국 일괄 도입
  - 전속 설계사 채널로 확장 (18만명 추가 TAM)
```

### 지역 확장 순서

```
[첫 파트너 지역] -> 경기 남부 (화성/수원/안산)
  -> 서울 서남부 (구로/금천/영등포)
  -> 인천 남동
  -> 서울 강남/서초 (고가치)
  -> 부산/경남 (2nd 거점)
  -> 대전/충청 (3rd 거점)
```

## 5. 진입 리스크 & 대응

| 리스크 | 심각도 | 대응 |
|--------|--------|------|
| **대형 GA 자체 CRM 구축** | 높음 | 중소형 GA 먼저 공략 -> 대형 GA는 "지점 단위" 침투. 비즈매칭은 대형 GA 자체 CRM에 없는 차별 기능 |
| **설계사 디지털 도구 거부감** | 중간 | 50-60대 설계사 비중 높음. UX 극단 단순화 + 카카오톡 연동 필수 |
| **DB 활용 규제 (개인정보)** | 높음 | 2024 개정 보험업법, 금소법 강화. 기존 DB 재활용 시 동의 갱신 필요. 법률 검토 필수 |
| **GA 이동성 (설계사 이탈)** | 중간 | 설계사가 GA를 옮기면 DB도 이동. SaaS를 설계사 개인 아닌 GA법인 단위로 계약 |
| **비즈매칭 초기 유동성 부족** | 높음 | Cold start 문제. 첫 GA의 50-100K DB로 초기 매칭풀 확보. 지역 상공회의소/소상공인연합 제휴로 보완 |
| **수도권 집중 리스크** | 낮음 | 시장의 55-60%가 수도권이므로 오히려 집중이 효율적. 지방은 Phase 4 |

---

## 6. 핵심 인사이트

1. **중소형 GA가 진짜 Beachhead**: 대형 GA (74개)는 자체 시스템 있어 진입 어렵고 협상력 약함. 중소형 GA (4,214개, 설계사 47,830명)는 도구 없이 설계사에게 "맨몸 영업"만 시키는 상태 -> AI CRM이 즉시 가치 제공
2. **비즈매칭이 진정한 차별화**: CRM만으로는 세일즈포스/자체개발 대비 경쟁력 약함. "보험 고객(사업주) 간 비즈니스 연결"은 기존 보험 CRM에 없는 기능 -> 설계사의 "관계 자산" 강화
3. **수도권 단일 거점 전략이 맞음**: 한국은 수도권에 인구/기업/GA가 과반 집중. 지역 분산 불필요. 수도권에서 500명 설계사 확보하면 PMF 검증 충분
4. **Co-founder의 GA 네트워크가 최대 자산**: GA 업계는 인적 신뢰 기반. 콜드콜/마케팅보다 네트워크 소개가 10x 효율적. 확장 속도를 결정하는 핵심 변수
