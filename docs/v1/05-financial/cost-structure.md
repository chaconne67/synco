## Break-Even Analysis

### Phase 2 기준 (Month 6~12)

```
월 고정비: 185만원
FC 변동비 (100명): 20만원
총 월 비용: 205만원

CEO 유료 공헌이익:
  Premium 구독: 9.9만 × 85% (PG+쉐어링 후) = 8.4만원/CEO
  크레딧 평균: 5만 × 85% = 4.25만원/CEO
  합산: ~12.65만원/유료 CEO

BEP = 2,050,000 ÷ 126,500 ≈ 17명 유료 CEO
```

> **유료 CEO 17명이면 월 손익분기.** CEO 500명 × 4% 전환 = 20명. 현실적. [Estimate]

### Phase 3 기준 (Year 2+)

```
월 고정비: 755만원
FC 변동비 (500명): 100만원
총 월 비용: 855만원

구독+크레딧 BEP = 8,550,000 ÷ 126,500 ≈ 68명 유료 CEO
+ 딜 중개 매출 (월 30건 × 플랫폼 몫 77만) = 2,310만원/월 추가 수익
```

> CEO 5,000명 × 4% = 200명 유료 + 딜 중개 매출. 구독만으로 BEP 3배 + 딜 중개가 추가 이익. [Estimate]
> **딜 중개 마진 분석:** 거래액 3,000만원 기준 수수료 210만원 → FC 60% 126만 + PG 3% 6.3만 → 플랫폼 순수익 77.7만원 (마진 37%) [Estimate]

## Flags

**Red Flags:** None

**Yellow Flags:**
- AI API 비용이 사용량 비례 상승 — FC 1,000명 시 월 200만원은 CEO 매출로 커버 필수
