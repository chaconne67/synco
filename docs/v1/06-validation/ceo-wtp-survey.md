# CEO WTP 검증 설문

**Project:** synco
**Date:** 2026-03-26
**Purpose:** CEO가 "AI 개인비서 CRM 앱"에 지불 의향이 있는지 검증
**Target:** 중소기업 CEO 30명+ (매출 10~500억, 업종 다양)
**Method:** 온라인 설문 (구글 폼 / 타입폼) + 고관심 응답자 심층 인터뷰

---

## 설계 원칙

1. **현재 행동(As-Is) → 문제 인식 → 솔루션 반응 → WTP** 순서로 퍼널 구성
2. **"쓰겠다"가 아니라 "얼마를 내겠다"를 묻는다** — WTP가 핵심
3. **앱은 하나, 기능은 역할별** — "개인비서"로 진입, 유휴 자산 가치 증폭이 과금 포인트
4. **사전 등록(연락처 남기기)이 가장 신뢰할 수 있는 행동 지표**

## 분석 프레임워크

### 핵심 지표 대시보드

| 지표 | 문항 | Strong Signal | Weak Signal | Kill Signal |
|------|------|---------------|-------------|-------------|
| 유휴 자산 보유율 | Q8 | 70%+ | 50~70% | <40% |
| 관리 불만도 | Q6 | 평균 ≤2.5 | 2.5~3.5 | ≥4.0 |
| 기회 탐색 불만도 | Q12 | 평균 ≤2.5 | 2.5~3.5 | ≥4.0 |
| 솔루션 관심도 | Q13 | 평균 3.5+ | 2.5~3.5 | <2.5 |
| "지금 당장" 비율 | Q13 5점 | 15%+ | 5~15% | <5% |
| 크레딧 WTP | Q17 | "3,000원+" 50%+ | 30~50% | <20% |
| 월 구독 적정가 | Q20 중앙값 | 5만원+ | 3~5만원 | <2만원 |
| 유료 전환 의향 | Q26 | "유료 고려" 40%+ | 20~40% | <15% |
| **사전 등록** | **Q27** | **20%+** | **10~20%** | **<5%** |
| 추천 의향 | Q28 | "5명+" 30%+ | 15~30% | <10% |

### Van Westendorp 분석 (Q19~Q22)

```
4개 곡선 교차점:
- PMC (Marginal Cheapness): "싸다" × "적절하다" → 가격 하한
- PME (Marginal Expensiveness): "비싸다" × "적절하다" → 가격 상한
- IDP (Indifference Price): "싸다" × "비싸다" → 최적 가격
- OPP (Optimal Price Point): 최적 수익 극대화 가격

→ 크레딧 단가(현 5,000원) / 구독가(현 9.9만원)의 적정성 판단
```

### 세그먼트 교차 분석

| 축 | 필터 | 분석 포인트 |
|----|------|-----------|
| 업종별 | Q2 | 제조업 vs 서비스업 WTP 차이 |
| 매출별 | Q3 | 매출 규모 ↔ WTP 상관관계 |
| 유휴 자산 有 vs 無 | Q8 | 유휴 자산 보유자의 WTP 프리미엄 |
| 현재 관리 불만 높음 vs 낮음 | Q6 | 불만 ↔ 사전 등록 전환율 |
| 선호 과금 모델 | Q25 | 크레딧 vs 구독 vs 성과기반 세그먼트 크기 |
| 최우선 기능 | Q15 | "CRM 먼저" vs "매칭 먼저" 비율 → 앱 우선순위 |

### 의사결정 매트릭스

```
■ Go (MVP 개발):
  관심도 3.5+ AND 크레딧 WTP 50%+ AND 사전 등록 20%+
  → Van Westendorp IDP 기준 가격 설정

■ 조건부 Go (모델 조정):
  관심도 높으나 WTP 낮음
    → 무료 CRM 중심 + 광고/데이터 수익으로 피봇 검토
  WTP 높으나 관심 낮음
    → 컨셉 전달 문제 — 인터뷰로 보완
  구독 거부 + 성과기반 선호 강함
    → 딜 중개 수수료 모델 강화
  "인맥관리" 선호 > "매칭" 선호
    → CRM-first 진입, 매칭은 후순위 기능으로 전환

■ Kill (CEO WTP 가설 기각):
  관심도 <2.5 AND 유료 의향 <15% AND 사전 등록 <5%
  → 과금 모델 근본 재설계 또는 프로젝트 피봇
```

## 운영 가이드

### 배포
- **구글 폼** 또는 **타입폼** (모바일 최적화 필수)
- 예상 소요: **8~10분**
- 인센티브: 스타벅스 기프티콘 5,000원 (완료자 전원 or 추첨)

### 표본
- 최소 **30명**, 이상적 50명+
- 공동창업자 DB에서 업종/매출 다양하게
- FC 경유 배포 시 "내 설계사가 추천" 바이어스 주의

### 후속 행동
| 조건 | 행동 |
|------|------|
| Q13 4~5점 + Q27 "예" | → **심층 인터뷰 대상** (10명 목표) |
| Q17 "무료여야" + Q26 "관심 없다" | → **비채택 이유 분석** (Q30 집중) |
| Van Westendorp 결과 확보 | → **크레딧/구독 가격 확정** |
