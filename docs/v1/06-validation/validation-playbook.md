### Experiment 1: CEO 관심도 인터뷰 (Week 1)

**검증 가설:** CEO가 "다른 CEO의 유휴 자원과 내 자원의 시너지 매칭"에 관심을 가진다

**방법:**
1. 공동창업자 DB에서 CEO 20명 선정 (업종 다양하게)
2. 15분 전화/대면 인터뷰 진행
3. 스크립트:
   - "지금 사업에서 덜 쓰이는 설비나 자원이 있으신가요?"
   - "다른 분야 CEO와 사업 기회를 찾아본 적 있으신가요?"
   - "AI가 '이 CEO와 이런 협업이 가능합니다'라고 제안하면 관심 있으신가요?"
   - "그 정보를 보려면 월 5~10만원 정도면 어떠신가요?"
4. 반응을 5단계로 코딩: 무관심 / 약간 관심 / 관심 / 강한 관심 / "지금 당장 쓰고 싶다"

**측정 지표:**
- 관심 이상 응답 비율 (목표: 60%+)
- "지금 당장" 응답 비율 (목표: 15%+)
- 가격 저항점 (어느 금액에서 꺼리는지)

**성공 기준:** 20명 중 12명+ 관심, 3명+ "지금 당장"
**실패 기준:** 관심 6명 미만 → 문제 정의 자체 재검토
**비용:** 0원 (시간만 투입)
**소요 시간:** 3-5일

### Experiment 2: GA 파트너십 경유 FC의 CEO 소개 행동 검증 (Week 1, 병렬)

**검증 가설:** GA 파트너십으로 편입된 FC가 CEO에게 앱을 소개하고 가입시키는 행동을 한다

**방법:**
1. GA 파트너십 체결 후, GA가 소속 FC에게 앱 도입 안내
2. 편입된 FC 10명 대면 미팅:
   - "무료 AI 비서로 고객 관리해드립니다"
   - "고객 CEO가 우리 서비스에 가입하면 과금 수익의 15%를 드립니다"
   - "CEO 10명 Premium 가입 시 월 15만원 추가 수입"
3. 반응 및 우려 기록
4. 핵심 질문: "CEO에게 앱을 소개하고 가입 링크를 보내실 수 있나요?"

**측정 지표:**
- "소개하겠다" 응답 비율 (목표: 50%+)
- 예상 소개 CEO 수 (FC당 평균)
- GA 안내 효과 (신뢰 형성 여부)
- 주요 우려 사항 목록

**성공 기준:** 10명 중 5명+ "소개하겠다", FC당 평균 CEO 3명+
**실패 기준:** "소개하겠다" 3명 미만 → GA 인센티브 재설계 또는 FC 채널 전략 변경
**비용:** 0원 (대면 시간)
**소요 시간:** 2-3일

### Experiment 3: 랜딩페이지 + Fake Door 테스트 (Week 2)

**검증 가설:** CEO가 "비즈니스 매칭 서비스" 가치 제안에 가입 행동을 한다

**방법:**
1. 랜딩페이지 제작 (1일, Framer/Carrd)
   - 헤드라인: "당신의 유휴 자원으로 새 비즈니스를 찾아드립니다"
   - 3가지 핵심 가치 시각화 (매칭 예시 스크린샷 목업)
   - CTA: "무료로 시작하기" → 이메일/연락처 수집
2. FC 10명에게 링크 배포 요청 (Exp 2에서 "보내겠다"한 FC)
3. 2주간 데이터 수집

**측정 지표:**
- 랜딩페이지 방문 수 (FC당 전환율)
- 이메일 등록 수 (방문→등록 전환율, 목표: 20%+)
- 등록자 업종/규모 분포

**성공 기준:** FC 10명 경유 → 방문 100명+ → 등록 20명+
**실패 기준:** 등록 5명 미만 → 가치 제안 메시지 재설계
**비용:** 5~10만원 (도메인+랜딩페이지 툴)
**소요 시간:** 2주

### Experiment 4: 컨시어지 MVP — 수동 매칭 (Week 3-4)

**검증 가설:** CEO가 매칭 결과에 실제 가치를 느끼고, 추가 정보 열람에 과금 의향이 있다

**방법:**
1. Exp 3 등록자 중 CEO 10~20명에게 프로필 작성 요청 (구글 폼)
2. 수동으로 매칭 분석 (AI + 수작업)
   - CEO A의 유휴 설비 × CEO B의 신규 사업 니즈
   - 매칭 결과를 블러 처리한 PDF로 제공
3. "상세 정보를 보시려면 5,000원입니다" → 카카오페이/토스 간편 결제
4. 매칭 상세 열람 후 "연결을 원하시면 25,000원입니다"

**측정 지표:**
- 프로필 작성 완료율 (목표: 60%+)
- 블러 해제 결제율 (목표: 30%+)
- 연결 요청 결제율 (목표: 15%+)
- 매칭 품질 만족도 NPS

**성공 기준:** 상세 열람 결제 5건+, 연결 요청 2건+
**실패 기준:** 블러 해제 결제 2건 미만 → CEO WTP 가설 기각
**비용:** 5~10만원 (결제 수수료)
**소요 시간:** 2주

### Experiment 5: 가격 A/B 테스트 (Week 5-6)

**검증 가설:** CEO 최적 가격대가 크레딧 5,000원, Premium 월 9.9만원이다

**방법:**
1. Exp 4 이후 추가 CEO 모집 (목표 50명)
2. A/B 그룹:
   - Group A: 크레딧 3,000원 / Premium 5.9만원 (저가)
   - Group B: 크레딧 5,000원 / Premium 9.9만원 (기본가)
   - Group C: 크레딧 8,000원 / Premium 14.9만원 (고가)
3. 전환율과 ARPU 비교

**측정 지표:**
- 그룹별 유료 전환율
- 그룹별 ARPU
- 그룹별 NPS
- Revenue = 전환율 × ARPU (최적점 도출)

**성공 기준:** 하나의 가격대에서 전환율 3%+ & 월 ARPU 5만원+
**실패 기준:** 모든 가격대에서 전환율 1% 미만 → 과금 모델 근본 재설계
**비용:** 0원 (컨시어지 MVP 확장)
**소요 시간:** 2주

### Experiment 6: AI 매칭 엔진 프로토타입 (Week 5-8)

**검증 가설:** GPT/Claude API 기반 매칭이 수동 매칭 대비 80% 이상 품질을 달성할 수 있다

**방법:**
1. Exp 4의 수동 매칭 케이스를 골드 스탠다드로 설정
2. CEO 프로필 데이터 → GPT-4o/Claude API 매칭 프롬프트
3. AI 매칭 결과 vs 수동 매칭 결과 비교
4. 매칭 정밀도, 재현율, CEO 만족도 비교

**측정 지표:**
- 수동 매칭 대비 일치율 (목표: 80%+)
- CEO 만족도 (수동 vs AI, 5점 척도)
- 매칭 1건당 API 비용

**성공 기준:** 일치율 80%+, 만족도 차이 0.5점 이내, 건당 비용 500원 이하
**실패 기준:** 일치율 50% 미만 → 규칙 기반 하이브리드로 전환
**비용:** 5~20만원 (API 비용)
**소요 시간:** 2-3주

### Experiment 7: 파일럿 GA 확장 (Week 8-10)

**검증 가설:** 첫 번째 GA 외 다른 GA에서도 FC 온보딩과 CEO 전환이 재현된다

**방법:**
1. 2~3개 추가 GA에 동일 프로세스 적용
2. FC 가입 → CEO 전환 → 과금 퍼널 전체 측정
3. GA별 성과 편차 분석

**측정 지표:**
- GA별 FC 가입률
- GA별 FC→CEO 전환율
- GA별 CEO 과금 전환율
- 운영 비용 / GA

**성공 기준:** 2개+ GA에서 유사 성과, 유료 CEO 20명+ (BEP 도달)
**실패 기준:** 1개 GA에서만 작동 → 특수 케이스인지 범용적인지 분석
**비용:** 0원 (시간 투입)
**소요 시간:** 4주

## Flags

**Red Flags:** None

**Yellow Flags:**
- Exp 1(CEO 관심) 실패가 가장 치명적. Exp 2는 GA 파트너십으로 대체 가능하므로 R2 하향. 단, 양쪽 동시 실패 시 전체 모델 재검토 필요
- 컨시어지 MVP(Exp 4)에서 수동 매칭 품질이 CEO 기대에 못 미치면 AI 매칭은 더 어려움
- 가격 A/B(Exp 5)는 표본 50명이라 통계적 유의성 약함 — 방향성 참고용
