## Value Proposition Canvas

#### Jobs-to-be-Done
| Job | Type |
|-----|------|

#### Pains
| Pain | 강도 |
|------|------|
| 보험 얘기하면 안 받음/차단 — "스팸 인식" | Critical |
| 고객 3,000명인데 보험 외 줄 게 없음 | Critical |
| 지인 소진 후 신규 리드 불가 | Critical |

| 이직/기기변경 시 DB 소실 | Medium |

| Gain | 유형 |
|------|------|
| 기존 DB에서 새로운 수익 기회 발견 | Required |
| 고객이 먼저 연락하는 관계 | Desired |
| 보험 외 가치 제공으로 관계 전환 | Desired |
| 법적으로 안전한 고객정보 관리 | Expected |
| 어디서든 접근 가능한 고객 DB | Expected |

### Value Map: 우리 제품

#### Products & Services
1. **AI 영업 비서 CRM** — 고객 등록, 메모, 일정, 알림, 엑셀 임포트
2. **AI 기회 발굴** — 기존 DB 분석 → 크로스셀/재접촉 타이밍 추천
3. **가치 콘텐츠 자동 생성** — 고객 업종별 비즈니스 인사이트
4. **비즈니스 매칭 엔진** — CEO 간 비즈니스 연결 제안 (Phase 2)

#### Pain Relievers
| Pain | Reliever |
|------|----------|
| 스팸 인식/차단 | "보험 얘기가 아니라 사업에 도움 되는 정보 드리려고요" — AI가 고객별 맞춤 연락 사유 제안 |
| 보험 외 줄 게 없음 | 업종별 비즈니스 인사이트 콘텐츠 자동 생성 → 가치 있는 연락 |
| 인맥 고갈 | AI가 기존 DB에서 숨은 기회 발견 (생일, 갱신, 업종 변화 등) |
| 카톡+엑셀 한계 | 카톡급 쉬운 UX, 엑셀 임포트, 클라우드 동기화 |
| 정보 전송 불안 | 암호화된 전용 플랫폼, 개인정보 위탁 처리 구조 |
| DB 소실 | 클라우드 저장, 기기 변경 시 데이터 유지 |

#### Gain Creators
| Gain | Creator |
|------|---------|
| 새로운 수익 기회 | AI가 재접촉 타이밍, 크로스셀 기회, 매칭 가능성 자동 알림 |
| 고객이 먼저 연락 | 가치 콘텐츠 지속 제공 → "보험 아닌 것도 알려주시네" → 신뢰 전환 |
| 비즈니스 커넥터 인정 | CEO 간 소개 성사 → "이 사람은 보험만 파는 게 아니다" |

## One-Sentence Value Proposition

> **"보험설계사의 잠자는 CEO DB를 AI가 깨웁니다 — 고객 관리부터 비즈니스 연결까지."**

**대안 버전:**
- "보험 말고도 줄 게 생깁니다. AI가 찾아주는 고객별 맞춤 기회."
- "3,000명 고객을 카톡에서 꺼내세요. AI 비서가 영업 기회를 알려드립니다."
