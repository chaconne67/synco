## April Dunford Positioning Framework

### 1. Competitive Alternatives

우리가 없으면 고객은 어떻게 하는가:

| 대안 | 사용 비율 | 한계 |
|------|----------|------|
| **카카오톡 + 엑셀 + 수첩** | ~90% [Estimate] | 검색 불가, 유실 위험, 개인정보 위반, AI 분석 불가 |
| **보험사 앱 5~10개** | ~80% | 각 보험사별 분산, 고객 전체 뷰 불가, CRM 아님 |
| **지넥슨/퀸텟 GA ERP** | 대형 GA | 계약 처리 중심, 설계사 개인 CRM 아님, AI 없음 |
| **보케어** | 극소수 | 기본 CRM만, AI 없음, GA 단위 관리 없음 |
| **아무것도 안 함** | 높음 | 기존 방식 유지, 성과 정체 수용 |

> 핵심 인사이트: 가장 큰 경쟁자는 소프트웨어가 아니라 **카카오톡 습관** [Data, `raw/indirect-competitors.md`]

### 2. Unique Attributes

우리만 가진 것:

| Attribute | 설명 |
|-----------|------|
| **AI 영업 기회 발굴** | 기존 고객 DB 분석 → 재접촉 타이밍, 크로스셀 기회 자동 추천 |
| **고객별 가치 콘텐츠 생성** | AI가 고객 업종/관심사 기반 비즈니스 인사이트 자동 생성 |

| **보험 설계사 전용** | GA 영업 워크플로우에 맞춘 기능 (조회, 외근, 실적 정리) |
| **현직 설계사가 만든 제품** | 현장 감각 반영, 설계사 언어로 소통 |

### 3. Value (Attribute → Customer Benefit)

| Attribute | Value |
|-----------|-------|
| AI 기회 발굴 | 잠자는 DB에서 새 계약 기회를 발견 → **수입 증가** |
| 가치 콘텐츠 생성 | "보험 아닌 얘기"로 전화 가능 → **차단 안 당함, 관계 유지** |
| 비즈니스 매칭 | CEO에게 사업 인맥 제공 → **"보험쟁이"에서 "비즈니스 커넥터"로 위상 전환** |
| 카톡 수준 UX | 배우는 데 5분 → **바로 업무에 적용, 학습 비용 0** |
| 보험 전용 | 불필요한 기능 없음 → **내가 필요한 것만 있는 도구** |

### 4. Target Customer (누가 가장 많이 신경 쓰는가)

**Best-Fit Customer:**
- GA 소속 보험설계사
- 법인영업 중심, 경력 5~15년, 팀장급
- CEO DB 500~3,000명 보유
- "보험 외 줄 게 없다"는 좌절감을 느끼는 사람
- 중소형 GA (30~200명) 소속 — 자체 시스템 없음

**왜 이들이 가장 많이 신경 쓰는가:**
- DB가 많을수록 AI 분석 효과 극대화
- 법인영업일수록 CEO에게 비즈니스 가치 제공 니즈 절실
- 중소형 GA일수록 도구가 없어서 전환 장벽 최저 [Data, `competitor-landscape.md`]

### 5. Market Category

**선택: 기존 카테고리의 새로운 서브카테고리**

> **"보험설계사 전용 AI 영업 비서"**

왜 이 카테고리인가:
- "CRM"이라고 하면 Salesforce/HubSpot 떠올림 → 비싸고 복잡하다는 선입견
- "영업 비서"는 설계사에게 직관적 — "내 일을 도와주는 도구"
- "AI"는 차별화 + 시대적 흐름 (AI CRM CAGR 36.1%)
- "보험설계사 전용"은 타겟을 명확히 좁혀줌

**피해야 할 카테고리:**
- "인슈어테크" — 보험사 대상 느낌, 설계사가 자기 것으로 안 느낌
- "B2B 매칭 플랫폼" — 너무 추상적, 설계사가 이해 못 함
- "CRM" 단독 — 범용 제품과 혼동, 가격 비교 시작됨

## Positioning Statement

> **GA 보험설계사 중** 기존 CEO DB에서 새로운 영업 기회를 찾고 싶은 분들을 위해,
> 우리 제품은 **AI 영업 비서**입니다.
> 카카오톡+엑셀과 달리, **AI가 고객 DB를 분석해 재접촉 타이밍과 비즈니스 기회를 자동으로 찾아줍니다.**
> 현직 설계사가 직접 만들어서 **설계사의 일과에 딱 맞습니다.**
