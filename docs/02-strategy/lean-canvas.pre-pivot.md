## 2. Customer Segments

**타겟 고객:** GA 소속 보험설계사, 법인영업 중심, 경력 5~15년
- 30만명 중 법인영업 중심 ~10만명 [Estimate]
- 평균 51.4세, 여성 60%, 수수료 100% 기반

**Early Adopter:** 공동창업자 소속 GA의 설계사 30명
- 이미 구두 협의 완료, CEO DB 5~10만건 보유
- 디지털 도구에 상대적으로 개방적인 팀장급

---

## 3. Unique Value Proposition

> **"보험설계사의 AI 영업 비서 — 고객 관리부터 비즈니스 연결까지"**

기존 CRM과의 차이: 단순 고객 정리가 아니라, AI가 기존 고객 DB에서 새로운 영업 기회를 발견하고 고객에게 제공할 가치 콘텐츠를 자동 생성

---

## 4. Solution (Top 3 Features)

| 문제 | 솔루션 |
|------|--------|
| 가치 비대칭 | **AI 맞춤 가치 콘텐츠** — 고객 업종/관심사 기반 비즈니스 인사이트 자동 제공 (Phase 1), 비즈니스 매칭 제안 (Phase 2) |
| 도구 부재 | **카톡급 UX CRM** — 고객 등록/메모/일정/알림, 카카오톡 연동, 엑셀 임포트 |
| 인맥 고갈 | **AI 기회 발굴** — 기존 DB에서 크로스셀/재접촉 타이밍 자동 추천 |

## 9. Unfair Advantage

1. **공동창업자 현직 설계사** — 즉시 접근 가능한 GA 네트워크 + CEO DB 3~4천건 + 현장 감각
2. **대면 영업 채널** — 한국 B2B는 레퍼런스 드리븐, 설계사 키맨이 입소문 = 유일한 유통 채널 [Data, `raw/channels.md`]
3. **도메인 복합 전문성** — 보험 8년 + 개발 7년, 양쪽 언어를 모두 구사

> 이 3가지는 돈으로 살 수 없고 쉽게 복제 불가
