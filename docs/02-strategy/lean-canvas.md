## 1. Problem — 데이터에 맥락이 없다

**핵심 문제:** GA가 돈 주고 사는 기업정보는 숫자 나열일 뿐, FC가 활용할 수 없다. FC의 CEO 네트워크는 보험 판매 후 가치 = 0으로 방치된다. CEO가 가진 인맥/자원도 잠재 가치를 모른 채 놀고 있다. **데이터는 있는데 맥락이 없어서 아무도 가치를 못 얻고 있다.**

| # | 파생 문제 | 누구의 문제 | 기존 대안 |
|---|----------|------------|----------|
| 1 | **맥락 없는 기업정보** → FC는 "보험 하나 보시죠"밖에 못 함 → CEO는 차단 | FC + CEO | 지인 소개(~80%), BNI(약 135~405만원/년), 리멤버(30만/년) |
| 2 | **관계의 사장** — 5년+ 쌓은 CEO 네트워크가 보험 판매에만 소진, 양쪽 모두 가치를 못 얻음 | FC + CEO | 없음. 보험 제안 반복 → 차단 |
| 3 | **AI 도입 갭** — 관심은 극대인데 실제 도입 극소. 격차는 지수적으로 벌어지는데 체감 못 함 | FC + CEO | 카카오톡 메모, 엑셀, 수첩 |

## 3. Unique Value Proposition

> **CEO:** "AI가 당신이 가진 것의 진짜 가치를 발견합니다 — 놓치고 있던 사업 기회를 찾아주는 매칭 플랫폼. AI 개인비서(인맥·일정·브리핑)는 무료"
> **FC:** "가치 없던 기업정보가 AI로 맥락을 만나면 무기가 됩니다 — 무료 AI 개인비서로 고객을 관리하고, CEO 매칭 수익을 나눠 받는 새 비즈니스"

**핵심 차별화:**
- AI가 맥락을 붙여 가치 없던 데이터를 가치 있게 전환 — synco의 본질
- **AI 개인비서의 기본 효용(인맥·일정·브리핑)이 지속 사용의 원동력** — 매칭 이전에 단독 가치가 있어야 한다
- 무료 AI 개인비서 = 원천 가치 체험 → 지속 사용 → 매칭 설득력 확보
- FC 대면 신뢰 채널 — AI가 발굴하고 사람이 성사시키는 하이브리드
- "넣으면 나온다" 선순환 — 데이터 입력 ↑ → 매칭 품질 ↑ → 입력 동기 ↑
- many-to-many DB 관계에서 AI가 연결성을 발견 — 사람이 보지 못하는 패턴을 찾아냄

## 4. Solution

| 문제 | 솔루션 |
|------|--------|
| CEO 유휴 자원 방치 | **AI 매치메이킹** — CEO 사업 정보 분석 → 유휴 자원 활용 기회 발굴 + 파트너 매칭 |
| FC-CEO 관계 일회성 | **FC 무료 AI 개인비서(인맥·일정·브리핑)** + 수익 쉐어링 → FC가 "비즈니스 커넥터" 역할로 전환 |
| 도구 부재 (AI 도입 갭) | **공통 AI 개인비서(인맥·일정·브리핑, 무료)** + 역할별 추가 기능 — FC 모드: AI 영업 비서, CEO 모드: 인맥/사업 관리 → 무료이므로 높은 채택률 |

## 9. Unfair Advantage

1. **CEO 직통 DB** — 돈으로 복제 불가. GA에겐 추가 가치 0인 유휴 자산 → synco가 수익화
2. **FC 대면 전달 채널** — 콜드 스타트 해결. 웜 채널 이미 존재
3. **FC 인센티브 정렬** — 무료 AI 비서 + 수익 쉐어링 → 자발적 CEO 유입
4. **양면 네트워크 효과** — CEO↑ → 매칭 가치↑ → FC↑ → CEO↑
5. **도메인 복합** — 보험 8년 + 개발 7년 + 로보어드바이저 (Layer 3 첫 상품)
6. **즉시 검증** — 공동창업자 DB 3~4K로 Day 1 프로토타입 테스트
