## 4. 헤드헌팅 검색용 분류 체계

후보자 검색 시 "인서울", "SKY", "상위권" 등의 조건을 매칭하기 위한 분류.

### 그룹별 분류

| 그룹 | 대학 | 검색 태그 |
|------|------|----------|
| **SKY** | 서울대, 연세대, 고려대 | sky, 최상위, 명문 |
| **서성한** | 서강대, 성균관대, 한양대 | 상위, 명문 |
| **중경외시** | 중앙대, 경희대, 한국외대, 서울시립대 | 인서울 상위 |
| **건동홍숙이** | 건국대, 동국대, 홍익대, 숙명여대, 이화여대 | 인서울 중상위 |
| **국숭세단** | 국민대, 숭실대, 세종대, 단국대 | 인서울 |
| **과기특** | KAIST, POSTECH, UNIST, GIST, DGIST | 이공계 특수, 최상위 |
| **지거국** | 부산대, 경북대, 전남대, 전북대, 충남대, 충북대, 강원대, 제주대 | 지방거점국립, 지역명문 |

### "인서울" 범위 정의

헤드헌팅에서 "인서울" 조건은 통상 아래 대학을 포함:
- SKY + 서성한 + 중경외시 + 건동홍숙이 + 국숭세단 + 서울과기대 + 상명대 + 광운대 + 명지대 + 삼육대 등 서울 소재 4년제 대학
- KAIST/POSTECH은 서울 소재가 아니지만 "인서울급" 또는 그 이상으로 평가

## 5. 활용 방법

synco AI 검색에서 "인서울 출신 찾아줘" 요청 시:
1. 후보자의 educations.institution을 위 분류와 매칭
2. "SKY 출신" → 서울대/연세대/고려대 필터
3. "인서울" → 그룹별 분류 전체 대학 필터
4. "이공계 명문" → KAIST/POSTECH/서울대 공대/UNIST/GIST 필터
5. "해외 명문대" → 주요 해외 대학 목록과 educations.is_abroad=True 결합
