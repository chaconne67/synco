# t23: 퍼널 클릭 가능한 내비게이션

> **Phase:** 3 — 워크플로우 연결
> **선행 조건:** t19 (tabChanged 이벤트 시스템)

---

## 배경

현재 개요 탭에 퍼널(funnel) 시각화가 있다. 각 단계(컨택, 추천, 면접, 오퍼)의 카운트가 표시되지만, 클릭할 수 없어 해당 탭으로 바로 이동하려면 탭바를 사용해야 한다.

퍼널 단계를 클릭 가능한 링크로 변경하면, 개요 탭에서 관심 있는 단계를 바로 확인할 수 있다. 또한 "관심" 카운트를 퍼널에 추가하여, 컨택→추천 전환 가능 건수를 한눈에 파악할 수 있게 한다.

## 요구사항

### 퍼널 단계를 클릭 가능한 링크로 변경

각 퍼널 단계 클릭 시 해당 탭으로 전환한다:

```
컨택 (18) → 관심 (7) → 추천 (3) → 면접 (1) → 오퍼 (0)
  [탭]      [필터]     [탭]      [탭]      [탭]
```

- "컨택" 클릭 → 컨택 탭 로드
- "관심" 클릭 → 컨택 탭 (결과=관심 필터 적용)
- "추천" 클릭 → 추천 탭 로드
- "면접" 클릭 → 면접 탭 로드
- "오퍼" 클릭 → 오퍼 탭 로드

각 링크에 `hx-get`으로 해당 탭 파셜 URL을 지정하고, `hx-target="#tab-content"`, 성공 시 `tabChanged` 이벤트를 발행하여 탭바 활성 상태를 갱신한다.

### "관심" 카운트 추가

- `_build_overview_context()`의 퍼널 데이터에 `interested` 카운트를 추가한다.
- 기존 `contacts` 카운트에서 예정(RESERVED)은 제외한다.

### 컨택 탭 결과 필터 지원

- `project_tab_contacts()` 뷰에 `?result=관심` query parameter 처리를 추가한다.
- 퍼널에서 "관심" 클릭 시 해당 필터가 적용된 컨택 목록을 표시한다.

## 제약사항

- 퍼널 클릭 시 `tabChanged` 이벤트를 클라이언트 측에서 발행한다 (`hx-on::after-request`).
- t19의 탭바 이벤트 시스템에 의존한다.
