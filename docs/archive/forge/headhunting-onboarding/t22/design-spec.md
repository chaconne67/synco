# t22: 예정 목록에 "컨택 등록" 버튼 추가

> **Phase:** 3 — 워크플로우 연결
> **선행 조건:** 없음 (독립적 UI 변경)

---

## 배경

서칭 탭에서 "예정 등록"한 후보자가 컨택 탭의 예정 목록에 표시된다. 현재 예정 목록의 각 항목에 "결과 기록" 버튼이 있으나, 라벨이 워크플로우상의 의미("예정 → 컨택 등록")를 명확히 전달하지 못한다.

## 요구사항

### "결과 기록" 버튼을 "컨택 등록"으로 변경

- 예정 목록의 각 행에 있는 "결과 기록" 버튼의 라벨을 "컨택 등록"으로 변경한다.
- 버튼 스타일을 텍스트 링크에서 강조 버튼(bg-primary, text-white)으로 변경하여 더 눈에 띄게 한다.
- 기능은 동일: 클릭 시 해당 후보자가 프리필된 컨택 등록 폼을 인라인으로 연다.

### 변경 전후 비교

```
변경 전: [결과 기록]  (텍스트 링크 스타일)
변경 후: [컨택 등록]  (강조 버튼 스타일)
```

## 제약사항

- 기능 변경 없이 라벨과 스타일만 변경한다.
- 기존 `hx-get` URL과 `hx-target`은 유지한다.
