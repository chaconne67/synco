# Archive — 삭제해도 무방한 과거 기록

이 폴더는 synco 기획·검사·담금질 과정에서 쌓인 **과거 자료**의 보관소이다.

> **이 폴더는 언제든 통째로 삭제해도 된다.** 현재 시점의 사업·업무·구현 맥락은 모두 [../master/](../master/) 의 3개 마스터 문서에 독립적으로 담겨 있다. `docs/archive/`를 참조하지 않아도 synco를 완전히 이해·개발·운영할 수 있도록 의도적으로 설계되었다.

## 왜 보존만 되고 참조되지 않나

기획이 여러 차례 바뀌면서 아래 세 시대층의 자료가 공존하게 되었다:

1. **과거 (voice-first)** — 음성 채팅 중심 기획. 현재는 레거시로 격하됨
2. **중간 (plans/ 워크플로우)** — P01~P19 phase 기반 기획
3. **최신 (대시보드 + FINAL-SPEC)** — `assets/ui-sample/` 목업을 단일 진실 소스로 하는 대시보드-first 재정리

마스터 3종은 세 시대층을 모두 녹여 **최신 방향**으로 정렬된 내용만 담고 있다. 이 폴더의 파일들은 의사결정의 맥락을 되짚어보고 싶을 때 읽는 용도일 뿐이며, 현재의 단일 진실 소스가 아니다.

## 주의

- 이 폴더의 파일에 **새 내용을 추가하지 않는다.** 새 기획/결정은 마스터 문서를 업데이트한다.
- 마스터 문서는 이 폴더의 파일을 링크하지 않는다. 모든 필요한 내용이 마스터 자체에 있다.
- 이 폴더를 삭제해도 마스터 3종과 코드·assets/ui-sample 만으로 synco의 모든 맥락이 유지된다.
