## 2. 원인 분석

파일 내부 구조(xlsx = ZIP)를 분석한 결과, **보이지 않는 빈 도형(shape) 객체 59,457개**가 원인으로 확인되었습니다.

| 항목 | 값 |
|------|-----|
| 시트 | 2023-2025 (1개) |
| 데이터 범위 | 3,527행 × 17열 |
| 수식 | 0개 |
| 조건부 서식 | 0개 |
| 이미지 | 0개 |
| **도형(shape)** | **59,457개** |
| drawing1.xml 압축 해제 크기 | **61.9MB** |

### 왜 느려지는가

Excel은 화면을 그릴 때마다 모든 도형 객체의 위치, 크기, Z-order를 재계산합니다. 데이터는 3,500행으로 가볍지만, 6만 개 도형의 렌더링 연산이 매 프레임 발생하여 편집 속도가 극심하게 저하됩니다.

### 도형이 생긴 원인 (추정)

- 메모/댓글이 있는 셀 범위를 반복 복사-붙여넣기하면서 도형이 누적
- 다른 파일/시트에서 데이터를 복사할 때 도형이 함께 딸려옴
- 빈 텍스트 상자가 행 단위로 자동 생성된 경우

## 3. 처리 내용

openpyxl을 사용하여 데이터와 서식만 새 파일로 복사하고, 도형 객체는 전량 제거했습니다.

**복사 항목:**
- 셀 데이터 (값)
- 셀 서식 (글꼴, 배경색, 테두리, 정렬, 숫자 형식)
- 열 너비 / 행 높이
- 병합셀
- 자동 필터
- 틀 고정

**제거 항목:**
- 도형(shape) 59,457개
