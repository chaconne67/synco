# P04: Project Multi-View

> **Phase:** 4 / 6
> **선행조건:** P03 (project basic CRUD — 리스트 뷰 존재)
> **산출물:** 보드(칸반), 리스트(액션), 테이블(스프레드시트), 캘린더 — 4가지 뷰 전환

---

## 목표

프로젝트 목록을 노션 스타일 4가지 뷰로 전환 가능하게 한다.
모든 뷰는 동일한 필터를 공유하며, 뷰 탭 전환만으로 관점을 바꾼다.

---

## URL 설계

기존 `/projects/` URL에 `view` 파라미터 추가. 별도 URL 불필요.

| URL | 파라미터 | 설명 |
|-----|---------|------|
| `/projects/?view=board` | view=board | 칸반 보드 (기본값) |
| `/projects/?view=list` | view=list | 액션 중심 리스트 |
| `/projects/?view=table` | view=table | 스프레드시트 |
| `/projects/?view=calendar` | view=calendar | 캘린더 |

필터 파라미터(`scope`, `client`, `status`, `sort`)는 뷰 전환 시 유지.

## 추가 URL

| URL | Method | 설명 |
|-----|--------|------|
| `/projects/<uuid:pk>/status/` | PATCH | 칸반 드래그앤드롭 상태 변경 |
| `/projects/calendar-events/?year=&month=` | GET | 캘린더 이벤트 데이터 |

## 테스트 기준

| 항목 | 검증 방법 |
|------|----------|
| 뷰 전환 | 4개 탭 클릭 시 각각 다른 뷰 렌더링 |
| 필터 유지 | 뷰 전환 시 기존 필터 파라미터 보존 |
| 칸반 D&D | 카드 드래그 후 status 변경 DB 반영 확인 |
| 긴급도 분류 | 리스트 뷰에서 조건별 올바른 그룹 배치 |
| 테이블 정렬 | 컬럼 헤더 클릭 시 정렬 변경 |
| 캘린더 이벤트 | 면접일/재컨택일이 올바른 날짜에 표시 |
